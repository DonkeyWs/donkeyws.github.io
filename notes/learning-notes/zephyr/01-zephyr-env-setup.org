** 楔子(Prologue)

今年早些时候发现了 =sesame robot= 这个开源项目，使用的是 =ESP32= 开发版和一些打印零件，自己做了一个跟孩子玩，
硬件花半天时间就都搞定了，可是软件部分不那么顺利，项目作者使用了 =Arduino= ，但是我个人更倾向于 =ESP-IDF= ，
后来索性就想顺手学学 =Zephyr= ，就暂停了准备学习下 =Zephyr= ，之后软件部分以它为系统实现。

作为系统化学习 =Zephyr RTOS= 的第一篇文章，从基础的环境搭建和 =demo= 工程开始。

** 环境安装(Setup)

#+begin_note
这里只介绍 =macOS= 上的一种方案，因为笔者是在一台老旧的 =Macbook Pro 2015= 笔记本上搭建的环境，
此时最流行的报管理工具 =brew= 已经不在支持此设备，索性 =macport= 还可以用，所以这里就是用 =macport=
在一台 =x64= 架构的 =macOS= 上搭建环境。
#+end_note

#+begin_src bash
  # 安装 Xcode Command Line Tools
  xcode-select --install

  # 安装 macport，从如下页面下载合适的pkg，并安装
  https://github.com/macports/macports-base/releases

  # 安装必要的依赖工具
  sudo port install cmake ninja gperf python3 ccache qemu dtc wget
#+end_src
